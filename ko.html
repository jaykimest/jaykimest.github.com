Android 업데이트

  • Version 7.10.25

    = NEW =

    • Resending files
    - In 'History' menu, you can resend the files.
    - Tap each history and choose individual files. Click Send and it will go straight to the transfer screen.
  • Version 7.9.29

    = NEW =

    • New key Notification
    - When a new key is created from other devices, you'll be notified. (Login Required)
  • Version 7.8.30

    = NEW =

    • My Link
    - In every device you're logged in, access the links you created in the My Link.
    - You can renew or delete the link in here.

    • Support Social Login
    - You can now login using Facebook, Google accounts.

    • Sending files between my devices has never been easier!
    - In the receive tab, you can check the files sent from the other devices.
  • Version 7.4.19

    = NEW =

    • Now you can extend the expiration date of the Share Link!
    - Simply tap the Clock icon and the expiration date will be extended to 48 hours again.

    • A 'Recent' tab has been added!
    - Files that have been added during last 30 days will displayed, grouped by the time.
  • Version 7.1.4

    = NEW =

    • Now you can search for files to send!
    - Tap the magnifying glass icon at the top of the file selection screen.
    - You can search for photos, videos, audio, apps, contacts, and all the files in the SendAnywhere folder and download path folder.
    - Filter by file type and last modified date.
 
iOS 업데이트
24
  • Version 7.11.23

    = NEW =

    • HEIC -> JPG converting option (iOS11 or later)
    - HEIC files saved in Camera roll can be converted to JPG format when sending.
    - Go to More > Settings and enable ‘Send HEIC photos as JPG’.
  • Version 7.11.1

    = NEW =

    • Support Files app (iOS11 or later)
    - In the iPhone’s Files app, things saved in ‘Files’ tab can be checked and managed.
    - Changes made in Files app also affect ‘Files’ tab in Send Anywhere app. When changing the names of a file or folder, use the Files app!

    • New key generation alert
    - If you create a 6-digit key or a link In the other logged in devices, you’ll receive an alert.
    - Long press the notification and select ‘Receive’ to download it right away.
  • Version 7.9.27

    = NEW =

    ▶ Sign-in features

    • My Link (New!)
    - On every device you're logged in, access the links you created in the My Link.
    - You can renew or delete the link in here.

    • Support Social Login
    - You can now login using Facebook, Google accounts.

    • Sending files between my devices has never been easier!
    - In the receive tab, you can check the files sent from the other devices.
    - Download the file by clicking the icon right next to the key.
  • Version 7.7.12

    = Improved =

    • Optimized to iPad
    - User Interface is now optimized for iPad to better your experience.

    • Music Player

    = NEW =

    • Brand new VIDEO PLAYER!
    - Supports a lot of different video formats.
    - Subtitle supports, with selectable text encoding.
    - Now watch your favorite movies on the go!

    = Improved =

    • You can now drag left and right to select multiple photos at once.
    • Select videos by album
    • Other bug fixes and performance improvements
  • Version 7.2.28

    = NEW =

    Introducing the all-new Send Anywhere for iOS - a complete redesign for more intuitive, inspiring experience.

    • Photo Preview
    - Long press on the thumbnail image to preview while selecting photos to send.

    • Activity tab
    - See all your transfers at a glance.

    • Pause / Resume
    - Transfers that have been interrupted or failed to complete, can now be resumed later on. Also, when uploading and downloading files via Share Link, you can pause and resume within the expiration period.

    • New Music Player
    - Completely redesigned music player inside. Easily transfer your music files and play them right away.

    • Improved stability
    - We have worked hard to improve file transfer stability and speed. Experience the enhanced performance now.
